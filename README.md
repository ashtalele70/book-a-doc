# Instructions to run the application

# Pre-requisites

1. Install Node

# Backend

1. Go to book-a-doc/backend
2. Run "npm i"
3. Run "node index.js"

# Frontend

1. Go to book-a-doc/frontend
2. Run "npm i"
3. Run "npm start"
